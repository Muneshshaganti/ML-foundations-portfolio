# ML Foundations Portfolio

> Structured hands-on exercises covering core machine learning and data analysis — the foundation before building production RAG and LLM systems.

This repo documents the systematic study phase that built the intuition behind the more advanced AI projects in this profile. Every notebook is a working implementation, not just theory.

---

## What's covered

| Topic | Techniques |
|---|---|
| Exploratory Data Analysis | Distribution analysis, correlation, outlier detection, visualization |
| Feature Engineering | Encoding, scaling, imputation, feature selection |
| Supervised Learning | Logistic regression, decision trees, random forests, SVM, KNN |
| Regression | Linear, polynomial, regularization (Ridge, Lasso) |
| Model Evaluation | Cross-validation, confusion matrix, ROC/AUC, precision-recall |
| Data Pipelines | Pandas workflows, NumPy operations, scikit-learn pipelines |

---

## Stack

Python · Jupyter Notebooks · scikit-learn · Pandas · NumPy · Matplotlib · Seaborn

## Learning progression

This repo is part of a deliberate build-up:

```
ML Foundations (this repo)
    ↓
Book Recommendations  ──  recommender system (content-based + collaborative filtering)
    ↓
AI Career Coach       ──  LLM + LangChain + prompt engineering
    ↓
RAG Document Advisor  ──  cloud RAG, Chroma, hybrid search, OCR
    ↓
Rag_Ollama            ──  fully offline RAG, local LLM, DB persistence
    ↓
multirag              ──  production RAG: multi-model fallback, multilingual, hybrid retrieval
```

Understanding how data gets cleaned, transformed, and modeled at the fundamentals level is what makes the higher-level systems less magical and more debuggable.
